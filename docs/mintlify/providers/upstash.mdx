## Overview

| Property | Value |
|----------|-------|
| Provider ID | `upstash` |
| Compatible Engines | Redis |
| SSL Default | TLS enabled (`rediss://`) |

## Connection

Upstash provides serverless Redis with per-request pricing. Mori connects using the `rediss://` scheme (TLS-enabled Redis).

```bash
mori init --from "rediss://default:pass@us1-shining-condor-12345.upstash.io:6379"
```

<Note>
Upstash always uses TLS. The provider generates connection strings in `rediss://` format (note the double `s`), which signals TLS to the Redis client.
</Note>
