## Overview

| Property | Value |
|----------|-------|
| Provider ID | `planetscale` |
| Compatible Engines | MySQL |
| SSL Default | `require` (TLS always enabled) |

## Connection

PlanetScale uses MySQL wire protocol with mandatory TLS. Mori generates a MySQL DSN with `tls=true`.

```bash
mori init --from "mysql://user:pass@aws.connect.psdb.cloud:3306/mydb"
```

<Note>
PlanetScale connections always use TLS. The provider hardcodes `tls=true` in the DSN format: `user:pass@tcp(host:port)/database?tls=true`.
</Note>
