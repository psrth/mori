## Overview

| Property | Value |
|----------|-------|
| Provider ID | `cloudflare` |
| Compatible Engines | SQLite, Redis |
| SSL Default | N/A (HTTPS API) |

## Connection

Cloudflare D1 (SQLite at the edge) and Workers KV are accessed via the Cloudflare API. You must provide your account and database IDs.

## Extra Fields

| Field | Description | Required |
|-------|-------------|----------|
| `cf_account_id` | Cloudflare Account ID | Yes |
| `cf_database_id` | D1 Database ID | Yes |

The provider constructs the API endpoint:
```
https://api.cloudflare.com/client/v4/accounts/<account_id>/d1/databases/<database_id>
```

## Example

```bash
# Interactive setup
mori init
# Select engine: sqlite
# Select provider: Cloudflare D1 / KV
# Enter Account ID and Database ID when prompted
```
